# CryptoHub

專業的加密貨幣交易平台，整合 AI 智慧分析、自動化交易策略和全方位投資組合管理。

## 概述

CryptoHub 將前沿人工智慧與量化交易深度結合，為交易者提供競爭優勢。平台支援加密貨幣、股票和外匯市場的即時分析，統一在一個介面中呈現。

### 核心功能

- **AI 智慧分析** — 機器學習模型分析市場數據、產生交易訊號，並提供帶置信度評分的建議。
- **策略引擎** — 使用 Python 腳本引擎撰寫、編譯、回測和部署交易策略，支援完整的事件驅動生命週期（`on_init` / `on_bar`）。
- **即時交易** — 透過 WebSocket 連接主流交易所，實現即時下單和組合追蹤。
- **回測引擎** — 在歷史數據上模擬策略執行，包含佣金和滑點建模，查看夏普比率、索提諾比率、最大回撤等多項指標。
- **多市場數據** — 內建加密貨幣（ccxt/Binance）、股票（Yahoo Finance）和外匯數據提供者。
- **投資組合管理** — 監控風險、追蹤未實現損益，當回撤閾值被突破時發送警示。
- **預測市場** — 整合 Polymarket 資料，用於情緒感知分析。
- **國際化** — 完整 i18n 支援，涵蓋英文、簡體中文、繁體中文、日文、韓文、阿拉伯文、俄文和德文。

## 快速開始

```bash
git clone https://github.com/louqiang1982/CryptoHub.git
cd CryptoHub
cp .env.example .env   # 編輯並填入你的 API 金鑰
docker compose up -d
```

| 服務 | 位址 |
|------|------|
| 前端 | http://localhost:3000 |
| Go API | http://localhost:8080 |
| Python API | http://localhost:8000 |
