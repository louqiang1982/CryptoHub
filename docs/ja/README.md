# CryptoHub

AI を活用した分析、自動売買戦略、包括的なポートフォリオ管理を備えた、プロフェッショナルな暗号資産取引プラットフォームです。

## 概要

CryptoHub は最先端の人工知能と定量取引を組み合わせ、トレーダーに競争上の優位性を提供します。暗号資産、株式、外国為替市場のリアルタイム分析を、統一されたインターフェースでサポートします。

### 主な機能

- **AI 分析** — 機械学習モデルが市場データを分析し、取引シグナルを生成、信頼度スコア付きの推奨を提供。
- **戦略エンジン** — Python ベースのスクリプトエンジンで取引戦略を作成・コンパイル・バックテスト・デプロイ。イベント駆動型ライフサイクル（`on_init` / `on_bar`）を完全サポート。
- **リアルタイム取引** — WebSocket 経由で主要取引所に接続し、ライブ注文実行とポートフォリオ追跡を実現。
- **バックテストエンジン** — 手数料とスリッページを考慮した歴史データでの戦略シミュレーション。シャープレシオ、ソルティノレシオ、最大ドローダウンなど多数の指標を算出。
- **マルチマーケットデータ** — 暗号資産（ccxt/Binance）、株式（Yahoo Finance）、外国為替の組み込みプロバイダー。
- **ポートフォリオ管理** — リスク監視、含み損益追跡、ドローダウン閾値超過時のアラート。
- **予測市場** — Polymarket データを統合したセンチメント分析。
- **国際化** — 英語、簡体字中国語、繁体字中国語、日本語、韓国語、アラビア語、ロシア語、ドイツ語の 8 言語に完全対応。

## クイックスタート

```bash
git clone https://github.com/louqiang1982/CryptoHub.git
cd CryptoHub
cp .env.example .env   # API キーを編集
docker compose up -d
```

| サービス | URL |
|---------|-----|
| フロントエンド | http://localhost:3000 |
| Go API | http://localhost:8080 |
| Python API | http://localhost:8000 |
