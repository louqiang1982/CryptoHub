# CryptoHub

منصة تداول عملات رقمية احترافية تتضمن تحليلات مدعومة بالذكاء الاصطناعي واستراتيجيات تداول آلية وإدارة شاملة للمحافظ الاستثمارية.

## نظرة عامة

يجمع CryptoHub بين الذكاء الاصطناعي المتقدم والتداول الكمي لمنح المتداولين ميزة تنافسية. تدعم المنصة التحليل الفوري لأسواق العملات الرقمية والأسهم والعملات الأجنبية من واجهة موحدة.

### الميزات الرئيسية

- **تحليلات الذكاء الاصطناعي** — نماذج تعلم آلي تحلل بيانات السوق وتولّد إشارات تداول مع تقييم الثقة.
- **محرك الاستراتيجيات** — كتابة وتجميع واختبار ونشر استراتيجيات التداول باستخدام Python مع دورة حياة كاملة (`on_init` / `on_bar`).
- **التداول الفوري** — اتصال WebSocket بالبورصات الكبرى لتنفيذ الأوامر الحية وتتبع المحفظة.
- **محرك الاختبار الخلفي** — محاكاة الاستراتيجيات على البيانات التاريخية مع نمذجة العمولات والانزلاق السعري، مع حساب نسبة شارب وسورتينو والحد الأقصى للتراجع.
- **بيانات متعددة الأسواق** — موفرو بيانات مدمجون للعملات الرقمية (ccxt/Binance) والأسهم (Yahoo Finance) والفوركس.
- **إدارة المحافظ** — مراقبة المخاطر وتتبع الأرباح والخسائر غير المحققة والتنبيه عند تجاوز حدود التراجع.
- **أسواق التنبؤ** — دمج بيانات Polymarket للتحليل المبني على المشاعر.
- **التدويل** — دعم كامل لثماني لغات: الإنجليزية والصينية المبسطة والتقليدية واليابانية والكورية والعربية والروسية والألمانية.

## البدء السريع

```bash
git clone https://github.com/louqiang1982/CryptoHub.git
cd CryptoHub
cp .env.example .env   # عدّل مفاتيح API
docker compose up -d
```

| الخدمة | الرابط |
|--------|--------|
| الواجهة الأمامية | http://localhost:3000 |
| Go API | http://localhost:8080 |
| Python API | http://localhost:8000 |
