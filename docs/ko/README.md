# CryptoHub

AI 기반 분석, 자동 매매 전략, 포괄적인 포트폴리오 관리를 갖춘 전문 암호화폐 거래 플랫폼입니다.

## 개요

CryptoHub은 최첨단 인공지능과 퀀트 트레이딩을 결합하여 트레이더에게 경쟁 우위를 제공합니다. 암호화폐, 주식, 외환 시장의 실시간 분석을 하나의 통합 인터페이스에서 지원합니다.

### 주요 기능

- **AI 분석** — 머신러닝 모델이 시장 데이터를 분석하고, 거래 신호를 생성하며, 신뢰도 점수가 포함된 추천을 제공합니다.
- **전략 엔진** — Python 기반 스크립팅 엔진으로 거래 전략을 작성, 컴파일, 백테스트, 배포합니다. 이벤트 기반 라이프사이클(`on_init` / `on_bar`)을 완전 지원합니다.
- **실시간 거래** — WebSocket을 통해 주요 거래소에 연결하여 라이브 주문 실행과 포트폴리오 추적이 가능합니다.
- **백테스트 엔진** — 수수료와 슬리피지를 고려한 과거 데이터 기반 전략 시뮬레이션. 샤프 비율, 소르티노 비율, 최대 낙폭 등 다양한 지표를 산출합니다.
- **멀티 마켓 데이터** — 암호화폐(ccxt/Binance), 주식(Yahoo Finance), 외환 내장 데이터 제공자.
- **포트폴리오 관리** — 리스크 모니터링, 미실현 손익 추적, 낙폭 임계치 초과 시 알림.
- **예측 시장** — Polymarket 데이터를 통합한 센티먼트 분석.
- **국제화** — 영어, 간체 중국어, 번체 중국어, 일본어, 한국어, 아랍어, 러시아어, 독일어 8개 언어 완전 지원.

## 빠른 시작

```bash
git clone https://github.com/louqiang1982/CryptoHub.git
cd CryptoHub
cp .env.example .env   # API 키 편집
docker compose up -d
```

| 서비스 | URL |
|--------|-----|
| 프론트엔드 | http://localhost:3000 |
| Go API | http://localhost:8080 |
| Python API | http://localhost:8000 |
