# CryptoHub

Профессиональная платформа для торговли криптовалютами с аналитикой на базе ИИ, автоматизированными торговыми стратегиями и полноценным управлением портфелем.

## Обзор

CryptoHub объединяет передовой искусственный интеллект с количественной торговлей, давая трейдерам конкурентное преимущество. Платформа поддерживает анализ рынков криптовалют, акций и валют в реальном времени — всё в едином интерфейсе.

### Ключевые возможности

- **ИИ-аналитика** — модели машинного обучения анализируют рыночные данные, генерируют торговые сигналы и предоставляют рекомендации с оценкой уверенности.
- **Движок стратегий** — создание, компиляция, бэктестинг и развёртывание торговых стратегий с помощью Python-скриптового движка с полным событийным жизненным циклом (`on_init` / `on_bar`).
- **Торговля в реальном времени** — подключение к крупным биржам через WebSocket для исполнения ордеров и отслеживания портфеля.
- **Движок бэктестинга** — симуляция стратегий на исторических данных с моделированием комиссий и проскальзывания; расчёт коэффициента Шарпа, Сортино, максимальной просадки и других метрик.
- **Мультирыночные данные** — встроенные провайдеры для криптовалют (ccxt/Binance), акций (Yahoo Finance) и форекса.
- **Управление портфелем** — мониторинг рисков, отслеживание нереализованной прибыли/убытка, оповещения при превышении пороговых значений просадки.
- **Рынки прогнозов** — интеграция данных Polymarket для анализа на основе настроений.
- **Интернационализация** — полная поддержка 8 языков: английский, упрощённый и традиционный китайский, японский, корейский, арабский, русский и немецкий.

## Быстрый старт

```bash
git clone https://github.com/louqiang1982/CryptoHub.git
cd CryptoHub
cp .env.example .env   # укажите свои API-ключи
docker compose up -d
```

| Сервис | URL |
|--------|-----|
| Фронтенд | http://localhost:3000 |
| Go API | http://localhost:8080 |
| Python API | http://localhost:8000 |
